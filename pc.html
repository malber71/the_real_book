IMPORTANT:
Adobe® Reader® 9 is included on this disc and must be installed for this product to work properly. If you have an older version of Adobe Reader, please uninstall it and then install Adobe Reader 9. If you have a newer version of Adobe Reader, then leave that version installed. This product works with newer versions of Adobe Reader, as of the time of production. If you have problems with the Adobe Reader, contact Adobe. DO NOT try to install different versions of Adobe Reader on the same computer at the same time.

Setting up Autorun:
  • This CD-ROM should startup automatically. If it does not, or if Autorun is disabled, then choose Start, Run and enter "D:\start.html" ("D" represents your CD-ROM drive) to directly open the CD-ROM Start HTML page.
  • When ready to browse the CD-ROM, make sure you are back on the CD-ROM Start HTML page and click the button that says "Contents". Adobe Reader should start up and you should navigate directly to the Table of Contents for this CD-ROM.
